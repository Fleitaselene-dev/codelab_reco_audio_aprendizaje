Reconocimiento de audio mediante aprendizaje por transferencia

Haz un sonido distintivo para cada botón

Por ejemplo: Manten precionado “Left” por 10s mientras silvas.



Entrena el modelo

Luego de grabar sonidos para cada botón, presiona el siguiente
botón para entrenar el modelo. Durante el entrenamiento, los
botones estarán deshabilitados temporalmente.



Escucha en tiempo real

Repite los sonidos, el modelo los reconocera y te los mostrara
en el siguiente formato:

    Las etiquetas para las clases son números enteros:
  • 0 = Left
  • 1 = Right
  • 2 = Noise
